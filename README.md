# Data Analytics Projects

This repository contains my Data Analytics internship projects completed as part of the DecodeLabs Industrial Training Program. The projects demonstrate the complete workflow of preparing and analyzing data using Python.

## Project 1: Data Cleaning & Preparation

### Objective
Prepare the raw dataset for analysis by improving its quality and consistency.

### Tasks Performed
- Imported and explored the dataset.
- Identified and handled missing values.
- Removed duplicate records.
- Corrected data types.
- Checked data consistency.
- Prepared a clean dataset for analysis.

### Tools Used
- Python
- Pandas
- NumPy

---

## Project 2: Exploratory Data Analysis (EDA)

### Objective
Analyze the cleaned dataset to discover patterns, trends, relationships, and potential outliers.

### Tasks Performed
- Dataset overview and structure analysis.
- Descriptive statistics.
- Distribution analysis using histograms.
- Outlier detection using boxplots.
- Correlation analysis with a heatmap.
- Categorical data analysis using count plots.
- Summary of key insights and observations.

### Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

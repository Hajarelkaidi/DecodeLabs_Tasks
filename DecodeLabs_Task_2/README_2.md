# Project 2: Exploratory Data Analysis (EDA)

## Introduction
After completing the data cleaning process in Project 1, this project explores the cleaned dataset to better understand its characteristics. The analysis includes descriptive statistics, data visualization, outlier detection, correlation analysis, and key insights.

## Tasks Completed
- Dataset overview
- Descriptive statistics
- Distribution analysis
- Outlier detection
- Correlation analysis
- Categorical variable analysis
- Summary of findings

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Outcome
The exploratory analysis revealed important patterns, relationships, and trends in the dataset, providing a solid foundation for future predictive modeling and business analysis.

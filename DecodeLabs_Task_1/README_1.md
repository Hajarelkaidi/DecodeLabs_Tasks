# Project 1: Data Cleaning & Preparation

## Introduction
This project focuses on cleaning and preparing a raw dataset for analysis. The goal is to improve data quality by handling missing values, removing duplicates, correcting data types, and ensuring consistency before performing any analysis.

## Tasks Completed
- Imported the dataset
- Explored the dataset structure
- Handled missing values
- Removed duplicate records
- Corrected data types
- Checked data quality
- Prepared the final cleaned dataset

## Technologies Used
- Python
- Pandas
- NumPy
- Jupyter Notebook

## Outcome
The dataset was successfully cleaned and prepared for the next stage: Exploratory Data Analysis (EDA).
